Created by Rutgers Coding Bootcamp Students: Sean Conaty, Christian Eckenrode, and Daniel Petro
BitCurrents is dedicated to providing real-time and historical Bitcoin data through a variety of open-source applications. As students enrolled in the Rutgers Coding Bootcamp, BitCurrents was created as a group front-end project showcasing the web development tools we have learned so far.
Programming languages used: HTML, CSS, Javascript, jQuery, Ajax, Bootstrap, and MaterializeCSS.
API's used: Coindesk, The New York Times, Reddit, Google Charts, Twitter and Youtube widgets.
